## Key Features

- **RWA-Backed Borrowing** - Use tokenized stocks (e.g., **TSLAx, AAPLx**) and other RWAs as collateral to borrow **USDC/USDT**.
- **Unified RWA Lending Aggregator** - Compare and access lending markets across Solana DeFi protocols (Kamino, Jupiter, Drift, and more) in one place.
- **SOCL Risk Engine** - A smart liquidation mechanism that **protects against sudden liquidations** by gradually selling collateral and preserving user equity.
- **Native Liquidity Vaults** - Provide liquidity to earn yield from interest, liquidation spreads, and protocol rewards.
- **Future RWA Expansion** - Support planned for tokenized **gold, real estate, fixed-income assets,** and other institutional-grade RWAs.
